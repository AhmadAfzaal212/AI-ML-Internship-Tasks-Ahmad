# 🌸 Task 1: Iris Dataset Exploration

## 🎯 Objective

Analyze and visualize the Iris dataset to understand feature relationships and data distributions.

## 🛠️ Work Done

* Loaded dataset using pandas & seaborn
* Performed data inspection (`.head()`, `.info()`, `.describe()`)
* Created visualizations:

  * Scatter Plot
  * Histogram
  * Box Plot
  * Pair Plot
  * Heatmap

## 📊 Key Insights

* Dataset contains 150 samples and 5 columns
* No missing values found
* Strong relationship between petal length & width
* Setosa species is clearly separable

# ❤️ Task 3: Heart Disease Prediction

## 🎯 Objective

Build a classification model to predict heart disease risk.

## 🛠️ Work Done

* Cleaned and preprocessed dataset
* Handled categorical features and missing values
* Performed Exploratory Data Analysis (EDA)
* Trained models:

  * Logistic Regression
  * Random Forest (Best Model)

## 📈 Evaluation

* Accuracy Score
* Confusion Matrix
* Classification Report

## 📊 Key Insights

* Age, blood pressure, and glucose are major risk factors
* Random Forest achieved better performance
* Model can assist in early disease risk detection

# 🏠 Task 6: House Price Prediction

## 🎯 Objective

Predict house prices using property features.

## 🛠️ Work Done

* Generated and preprocessed dataset
* Encoded categorical variable (location)
* Applied feature scaling
* Trained models:

  * Linear Regression
  * Random Forest Regressor (Best Model)

## 📈 Evaluation

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## 📊 Key Insights

* Area, location, and bedrooms are key factors
* Urban properties have higher prices
* Random Forest provides more accurate predictions
